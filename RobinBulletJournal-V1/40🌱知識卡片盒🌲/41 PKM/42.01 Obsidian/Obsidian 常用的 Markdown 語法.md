# 流程圖

語法
[Flowcharts - Basic Syntax | Mermaid](https://mermaid.js.org/syntax/flowchart.html)

```
mermaid 
graph TD

    1[寫靈感筆記] --> 3

    2[寫文獻筆記] --> 3

    3[寫永久筆記]

    --> 4(永久筆記放入筆記盒)

    --> 5[從筆記盒發展想法]

    --> 6[挑出主題]

    --> 7[筆記轉為草稿]
    
    --> 8[發表作品]
```

呈現
```mermaid 
graph TD

    1[寫靈感筆記] --> 3

    2[寫文獻筆記] --> 3

    3[寫永久筆記]

    --> 4(永久筆記放入筆記盒)

    --> 5[從筆記盒發展想法]

    --> 6[挑出主題]

    --> 7[筆記轉為草稿]
    
    --> 8[發表作品]

```
